## ✅ 필수 테스트 항목 (우선순위 높음)

### 1. 지원 공고 (📋 regular) - 무료
**테스트 계정**: 일반 사용자

#### 1-1. 공고 생성
```
1. 로그인 (일반 사용자)
2. 구인공고 작성 페이지 이동
3. 공고 타입: "지원 공고" 선택
4. 필수 정보 입력
   - 제목: "테스트 지원 공고"
   - 위치: "서울시 강남구"
   - 급여: 시급 15,000원
   - 모집 인원: 3명
5. [공고 등록] 버튼 클릭
```

**기대 결과**:
- ✅ 공고 생성 성공 (무료이므로 칩 차감 없음)
- ✅ 구인공고 목록에 즉시 표시
- ✅ 📋 아이콘 표시
- ✅ 칩 비용 배지 없음

#### 1-2. 날짜 슬라이더 필터링
```
1. 구인공고 페이지 > "지원 공고" 탭 선택
2. 날짜 슬라이더 확인
   - "전체" 버튼 있는지 확인
   - "오늘" 버튼 있는지 확인
   - 내일 ~ +14일 날짜 버튼들 있는지 확인
3. "오늘" 클릭 → 오늘 날짜 공고만 표시되는지 확인
4. 내일 날짜 클릭 → 내일 날짜 공고만 표시되는지 확인
5. "전체" 클릭 → 모든 공고 다시 표시되는지 확인
```

**기대 결과**:
- ✅ 날짜 슬라이더가 "지원 공고" 탭에만 표시
- ✅ 날짜 클릭 시 해당 날짜 공고만 필터링
- ✅ "전체" 클릭 시 필터 해제

---

### 2. 고정 공고 (📌 fixed) - 유료
**테스트 계정**: 칩이 충분한 사용자

#### 2-1. 공고 생성 (7일 고정)
```
1. 로그인 (칩 보유 사용자)
2. 구인공고 작성 페이지 이동
3. 공고 타입: "고정 공고" 선택
4. 고정 기간: "7일" 선택
5. 필수 정보 입력
   - 제목: "테스트 고정 공고 7일"
6. [공고 등록] 버튼 클릭
```

**기대 결과**:
- ✅ 칩 차감 확인 모달 표시: "3 칩이 차감됩니다"
- ✅ 공고 생성 성공
- ✅ 📌 아이콘 표시
- ✅ "💰 3 칩" 배지 표시
- ✅ 만료일 배지 표시 (예: "D-7")

#### 2-2. 만료일 표시 확인
```
1. 구인공고 페이지 > "고정 공고" 탭 선택
2. 방금 생성한 공고의 만료일 배지 확인
   - D-7 표시 (파란색)
   - ⏰ 아이콘 있음
```

**기대 결과**:
- ✅ D-7 배지가 파란색으로 표시
- ✅ 만료 임박 시(D-3 이하) 노란색으로 변경
- ✅ 만료 시(D-0) 빨간색으로 변경

#### 2-3. 칩 부족 시 동작
```
1. 칩이 3개 미만인 계정으로 로그인
2. 공고 타입: "고정 공고" 선택
3. 고정 기간: "7일" 선택
4. [공고 등록] 버튼 클릭
```

**기대 결과**:
- ✅ 칩 부족 에러 메시지 표시
- ✅ 공고 생성 실패

---

### 3. 대회 공고 (🏆 tournament) - 무료 + 승인 필요
**테스트 계정**: 일반 사용자 + Admin

#### 3-1. 공고 생성 (일반 사용자)
```
1. 로그인 (일반 사용자)
2. 구인공고 작성 페이지 이동
3. 공고 타입: "대회 공고" 선택
4. 필수 정보 입력
   - 제목: "테스트 대회 공고"
   - 대회명: "2025 홀덤 챔피언십"
5. [공고 등록] 버튼 클릭
```

**기대 결과**:
- ✅ 공고 생성 성공 (무료이므로 칩 차감 없음)
- ✅ 🏆 아이콘 표시
- ✅ "승인 대기" 배지 표시 (노란색)
- ✅ 구인공고 목록에 표시되지 않음 (승인 전)

#### 3-2. Admin 승인 (Admin 계정)
```
1. 로그인 (Admin 계정)
2. 관리자 메뉴 > "대회 공고 승인" 페이지 이동 (/admin/job-posting-approvals)
3. 방금 생성한 "테스트 대회 공고" 찾기
4. [승인] 버튼 클릭
5. 승인 모달에서 [승인하기] 버튼 클릭
```

**기대 결과**:
- ✅ 승인 성공 메시지 표시
- ✅ 공고 상태가 "승인됨"으로 변경 (녹색 배지)
- ✅ 일반 사용자가 구인공고 목록에서 볼 수 있음
- ✅ 작성자에게 승인 알림 발송 (알림 센터 확인)

#### 3-3. Admin 거부 (Admin 계정)
```
1. 새로운 대회 공고 생성 (일반 사용자)
2. Admin 계정으로 승인 페이지 이동
3. [거부] 버튼 클릭
4. 거부 사유 입력 (10자 이상): "대회 정보가 불충분합니다"
5. [거부하기] 버튼 클릭
```

**기대 결과**:
- ✅ 거부 성공 메시지 표시
- ✅ 공고 상태가 "거부됨"으로 변경 (빨간색 배지)
- ✅ 작성자에게 거부 알림 발송 (거부 사유 포함)

#### 3-4. 알림 확인 (일반 사용자)
```
1. 공고 작성자 계정으로 로그인
2. 헤더의 알림 아이콘(🔔) 클릭
3. 알림 내용 확인
```

**기대 결과**:
- ✅ 승인 알림: "테스트 대회 공고" 공고가 승인되었습니다
- ✅ 거부 알림: "테스트 대회 공고" 공고가 거부되었습니다 + 거부 사유

### 4. 긴급 공고 (🚨 urgent) - 유료
**테스트 계정**: 칩이 5개 이상인 사용자

#### 4-1. 공고 생성
```
1. 로그인 (칩 5개 이상 보유 사용자)
2. 구인공고 작성 페이지 이동
3. 공고 타입: "긴급 공고" 선택
4. 필수 정보 입력
   - 제목: "테스트 긴급 공고"
5. [공고 등록] 버튼 클릭
```

**기대 결과**:
- ✅ 칩 차감 확인 모달 표시: "5 칩이 차감됩니다"
- ✅ 공고 생성 성공
- ✅ 🚨 아이콘 표시
- ✅ "💰 5 칩" 배지 표시
- ✅ **빨간 테두리 애니메이션** (깜빡임 효과)

#### 4-2. 시각적 강조 확인
```
1. 구인공고 페이지 > "긴급 공고" 탭 선택
2. 긴급 공고 카드의 테두리 확인
```

**기대 결과**:
- ✅ 빨간색 테두리
- ✅ 깜빡이는 애니메이션 (animate-pulse)
- ✅ 다른 공고보다 시각적으로 눈에 띔

### 5. 게시판 탭 구조 (5탭)
```
1. 구인공고 페이지 이동
2. 탭 목록 확인
```

**기대 결과**:
- ✅ 5개 탭 표시
  1. 지원 공고 (📋)
  2. 고정 공고 (📌)
  3. 대회 공고 (🏆)
  4. 긴급 공고 (🚨)
  5. 내 지원 현황
- ✅ 각 탭 클릭 시 해당 타입 공고만 필터링
- ✅ 날짜 슬라이더는 "지원 공고" 탭에만 표시
