### 시나리오 1: 보수적 추정 (Low)

#### 일일 공고 현황

## 🎯 핵심 인사이트

### ✅ 강점

1. **다양한 수익원**
   - 칩 소비, 구독, 충전, B2B 등 4가지 독립적 채널
   - 한 채널이 약해져도 다른 채널로 보완 가능

2. **지원 신청 = 주요 수익원**
   - 전체 칩 소비의 67.6%
   - 가장 안정적이고 예측 가능한 수익
   - 사용자 증가 시 선형적 증가

3. **확장성**
   - 사용자가 2배 증가하면 수익도 약 2배 증가
   - 인프라 비용은 선형 증가하지 않음 (Firebase 자동 확장)

4. **충전 패키지가 최대 수익원**
   - 전체의 40.2% (월 233만원)
   - "인기 패키지"가 가장 많이 팔림 (월 99만원)

### ⚠️ 고려사항

1. **일반 공고 무료**
   - 플랫폼 활성화용이지만 직접 수익 없음
   - 지원자 유입을 위한 전략적 무료화
   - **개선 제안**: 프리미엄 옵션 (상단 노출, 강조 표시 등)

2. **긴급공고 가격 검토**
   - 현재 5칩 (1,250원)이 적절한지 시장 반응 확인 필요
   - **개선 제안**: 시간대별 차등 가격 (당일 7칩, 3일 전 5칩)

3. **고정공고 칩 범위가 넓음**
   - 3~10칩의 편차가 크므로 명확한 기준 필요
   - **개선 제안**: 소형(3칩), 중형(5칩), 대형(10칩)으로 표준화

4. **B2B 비중이 낮음**
   - 현재 8.1%에 불과
   - **개선 제안**: 대형 체인점 영업 강화, 맞춤형 패키지 제공

### 💡 개선 제안

#### 1. 고정공고 칩 가격 표준화
```
소형 공고 (1-2개 지점):  3칩  (750원)
중형 공고 (3-5개 지점):  5칩 (1,250원)
대형 공고 (6-10개 지점): 10칩 (2,500원)
```

#### 2. 긴급공고 차별화 가격
```
당일 긴급:   7칩 (1,750원) +40%
1일 전:      6칩 (1,500원) +20%
3일 전:      5칩 (1,250원) 기본
```

#### 3. 일반공고 프리미엄 옵션
```
기본 등록:         무료 (현행 유지)
상단 고정 (24시간): +3칩 (750원)
강조 표시:         +2칩 (500원)
푸시 알림:         +1칩 (250원)
```

#### 4. B2B 대량 패키지
```
스타터 (월 30만원):   칩 150개 + 기본 지원
프로페셔널 (월 50만원): 칩 300개 + 우선 지원
엔터프라이즈 (월 100만원): 칩 무제한 + 전담 매니저
```
